# ImageViewer
**VR 기기에 최적화 되어있는 이미지 뷰어 앱 입니다**   

**사용된 아이콘 이미지들은 구글링을 통해 가져온것입니다.**
**앱을 상업적으로 이용하지 않습니다. (플레이스토어 출시x) 문제될시 삭제하겠습니다**
